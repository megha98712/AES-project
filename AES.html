AES

Advanced Encryption Standard

Block Size: 128 bits
Key Size: 128/192/256 bits
Rounds: 10/12/14

What is AES?

AES is a symmetric-key block cipher used worldwide. Supports 128/192/256-bit keys and replaced DES for higher security.

Key Features

  • Symmetric-key encryption
  • Block size 128 bits
  • Key size 128/192/256 bits
  • Rounds: 10/12/14 depending on key
  • Substitution-Permutation network

How AES Works

AES operates on 128-bit blocks with variable key sizes. It performs rounds of substitution, permutation, mixing, and key addition.
